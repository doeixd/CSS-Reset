Line Heights

  • Leading None (.leading-none): This text has no line height spacing, which makes it very compact but can be hard to read for longer text blocks.
  • Leading Tight (.leading-tight): This text has tight line height spacing, good for headings and short text blocks where space is at a premium.
  • Leading Snug (.leading-snug): This text has snug line height spacing, slightly more breathable than tight but still compact.
  • Leading Normal (.leading-normal): This text has normal line height spacing, good for most body text and general readability.
  • Leading Relaxed (.leading-relaxed): This text has relaxed line height spacing, providing more breathing room between lines for improved readability.
  • Leading Loose (.leading-loose): This text has loose line height spacing, providing maximum readability but takes up more vertical space.

Auto-Contrast Text System

The auto-contrast text system automatically determines optimal text color based on background color using OKLCH color space. Background utility classes set both the background-color and the --bg variable, which is used to calculate --auto-contrast-text.

How It Works

--auto-contrast-text: oklch(
  from var(--bg, currentColor)
  clamp(0.1, (var(--l-threshold, 0.65) / l - 1) * 999, 0.98)
  min(c, var(--c-threshold, 0.08))
  h
);
  • • Uses background lightness to determine text lightness
  • • Reduces chroma (saturation) for better readability
  • • Maintains the original hue for a harmonious look
  • • Has configurable thresholds for contrast preferences

Container Queries

Container queries allow components to adapt based on their container's size, not just the viewport. This enables truly responsive components that work in any layout context.

Named Breakpoints

:root {
  /* Standardized container query breakpoints */
  --cq-bp-xs: 20em;  /* ~320px */
  --cq-bp-sm: 30em;  /* ~480px */
  --cq-bp-md: 45em;  /* ~720px */
  --cq-bp-lg: 60em;  /* ~960px */
  --cq-bp-xl: 80em;  /* ~1280px */
  --cq-bp-2xl: 96em; /* ~1536px */
}

Usage

.cq-type-inline - Sets container-type: inline-size
.cq-type-size - Sets container-type: size
.cq-name-card - Sets container-name: card
@container card (min-width: var(--cq-bp-md)) { ... }

Enhanced Focus States

The framework includes an enhanced focus state system for better accessibility. Focus states are customizable and adapt to high contrast mode automatically.

Focus Variables

:root {
  --focus-ring-width: 2px;
  --focus-ring-offset: 2px;
  --focus-ring-color: var(--outline-focus);
  --focus-ring-style: solid;
  
  /* Form specific */
  --input-focus-ring-width: var(--focus-ring-width);
  --input-focus-ring-offset: 0px;
}
